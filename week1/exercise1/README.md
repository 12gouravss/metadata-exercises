# Week 1 – Exercise 1 – Metadata Creation

## Dataset

Credit Card Fraud Detection Dataset

## Source

Kaggle (Machine Learning Group, Université Libre de Bruxelles)

## Description

This dataset used contains credit card transactions from European cardholders. It is commonly used for fraud detection and anomaly detection tasks in machine learning.

## Objective

The goal of this assignment is to describe a dataset using two different metadata standards and understand how data is represented for different purposes.

## Metadata Used

DataCite XML: Used in research to describe datasets and assign DOIs for citation understandable by computers.
schema.org (JSON-LD): Used by search engines to understand and display datasets online

## Key Idea

The same dataset can be described in different ways depending on where it is used as academic systems (DataCite) or web/search systems (schema.org).
